Accessibility Statement

Scope:

This statement relates to all the web pages within this project.

Usage of website:

Magnifying content

It is possible to magnify text on all web pages up to approx. 300% without disrupting the layout. For instructions on how to increase magnification please see the appropriate guide for your platform/browser online.

Screen readers

Most screen readers, such as ChromeVox, are accounted for and will work just fine on this website. Although the navbar is initially hidden, it's elements, i.e. links to other web pages can all be properly selected regardless of its state.

Navigation

This website features an alternative navigation system that makes use of the html "accesskey" functionality. As long as you've got a physical keyboard connected, pressing keys 1-6 together with the correct activation key will navigate you to the desired page. For most browsers the activation key is "alt", however, for a complete list click HERE.
